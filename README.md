# 가위바위보 (Rock Paper Scissors)

브라우저에서 버튼을 클릭해 컴퓨터와 가위바위보를 즐기는 게임입니다.
The Odin Project의 RPS 프로젝트로 학습한 내용을 담았습니다.

🔗 **[라이브 데모 바로가기](https://jhoncarmack.github.io/Rock_Paper_Scissors_2026/)**

## 플레이 방법

- **주먹 / 보 / 가위** 버튼을 클릭해 한 라운드씩 진행합니다.
- 컴퓨터는 매 라운드 무작위로 선택합니다.
- 먼저 **5점**에 도달하는 쪽이 승리하며, 게임 종료 후 점수는 초기화됩니다.

## 화면 구성

- 가위바위보 선택 버튼
- 현재 점수 표시 (사용자 / 컴퓨터 / 총 경기 수)
- 게임 종료 시 승자 알림

## 사용 기술

- HTML
- CSS
- JavaScript (DOM 조작, 이벤트 위임)

## 배운 점

- `branch` / `merge` / `checkout`을 활용한 Git 브랜치 작업
- **이벤트 버블링**을 응용한 이벤트 위임 — 버튼마다 리스너를 다는 대신
  부모 요소(`p`)에 리스너 하나만 등록해 메모리를 절약
- `prompt` 입력 방식에서 버튼 클릭 UI로 코드 리팩토링

## 실행 방법

`index.html` 파일을 브라우저에서 열면 바로 플레이할 수 있습니다.
또는 위의 **라이브 데모** 링크에서 바로 플레이할 수 있습니다.
